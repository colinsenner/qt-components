# QT Components

## Windows Setup

- Install Qt 6.9.1 from the [Qt website](https://www.qt.io/download)
- Create `QTDIR` environment variable (e.g. `C:\Qt\6.9.1\msvc2022_64`)
- Add `%QTDIR%\bin` to your `PATH` environment variable
